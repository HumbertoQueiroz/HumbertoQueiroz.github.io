Projetos

Apliquei meus conhecimentos neste projetos a seguir:

  • Site Estacionamento

    Aplicação de conhecimento de HTML semântico, com metha tags como Keywords, description, robots e canonica! para um bom ranqueamento no SEO. CSS desenvolvido com Mobile force, site responsivo para todos os tipos de aparelhos, smartphone, tablet e desktop através de @media query.

  • Bateria digital

    Aplicação do conhecimento para criar funções, function, estrutura de repetição, for, estrutura condicional, if e else em JavaScript para manipular saudio dentro do HTML.

  • Barbearia

    Desenvolvimento de um site de Barbearia completo, aplicando boas práticas no html e css, desenvolvido com responsividade e compatibilidade web e mobile, com inserção de mídia dentro do html (vídeo, mapa).

  • Galeria de Imagens

    Aplicação de conhecimento de JavaScript para adicionar e alterar propriedades de elementos no html de forma automática em vários elementos através de uma estrutura de repetição com o for, acionamento de eventos com event handler/event listeners

  • Gerador de Histórias Bobas

    Utilização de Event Listener para manipular o html através do JavaScript

  • Jogo de Adivinhar Número

    Desenvolvimento de um jogo de adivinhar um número aleatório, gerado por random, aplicando JavaScript ao html para gerar, alterar e ocultar elementos.
