Conhecimentos

HTML semântico e metha tags para rankeamento de SEO, como Keywords para palavras chave, description para identificar o texto que vai aparecer no resutado da busca após o link do site, robots para indicar que deve ser indexado em buscas, canonical para definir o link canonico.

CSS para estilização do layout como animações, boxshadow, fontes, manipulação de elementos com o evento hover do mouse, position fixed para manter o menu no topo, display e visibility para mostar e esconder elementos e para responsividade utilizei Flexbox e @mediaquery.
Utilizado Metodologia BEM para nomeação e organização das classes.

JavaScript para manipulação de atributos e elementos do html com utilização de eventos como onclick para ativar funções, seletores como querySelector e GetElementById para selecionar elementos e alterar propriedades como display, visibility, color, fontWeight dando uma interatividade e tornando a página dinâmica, etc...

Framework BootStraps, para produtividade utilizando elementos e classes pré-prontos diminuindo o tempo de criação de um projeto.
